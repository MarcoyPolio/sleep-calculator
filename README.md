# 🌙 Sleep Hours Calculator

Free sleep calculator based on 90-minute sleep cycles.
Includes: By Age, Bedtime, Wake-Up, and Sleep Debt tabs.

---

## 🚀 Deploy to Vercel in 5 Steps

### Step 1 — Create a GitHub account
Go to https://github.com and sign up (free).

### Step 2 — Upload this project to GitHub
1. Click the **+** button → "New repository"
2. Name it: `sleep-calculator`
3. Set it to **Public**
4. Click "Create repository"
5. Upload ALL files from this folder by dragging them in

### Step 3 — Connect to Vercel
1. Go to https://vercel.com and sign up with your GitHub account
2. Click **"Add New Project"**
3. Select your `sleep-calculator` repository
4. Vercel auto-detects Vite — click **Deploy**
5. Wait ~60 seconds → your site is LIVE 🎉

### Step 4 — Add your custom domain
1. In Vercel dashboard → your project → **Settings → Domains**
2. Type your domain name and click Add
3. Follow the DNS instructions (Vercel walks you through it)

### Step 5 — Apply for Google AdSense
1. Go to https://adsense.google.com
2. Sign up with your Google account
3. Enter your live domain URL
4. Paste the verification code into index.html (the commented line)
5. Wait for approval (1–4 weeks)
